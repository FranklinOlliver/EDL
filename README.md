# 📚 EDL - Estruturas de Dados Lineares
Repositório criado para a disciplina de **Estrutura de Dados Lineares (EDL)**, contém implementações de diversas estruturas de dados em **Java**, desenvolvidas durante a disciplina. Abrangindo desde estruturas simples, como listas e filas, até estruturas mais complexas, como árvores.

## 🛠 Tecnologias
- Java JDK 17 ou superior

## 📂 Organização
Cada estrutura de dados está em sua própria pasta, contendo os arquivos Java necessários para a implementação. Algumas pastas também contêm exemplos de testes e uso das estruturas. Além disse, ainda contém uma pasta com os materiais da disciplina, dois arquivos .txt referentes a duas listas de exercícios e a apresentação do seminário final.

## 🚀 Como executar
1. Clone o repositório:
   ```bash
   git clone https://github.com/FranklinOlliver/EDL.git
    ```
2. Navegue até a pasta da estrutura de dados que deseja executar:
    ```bash
    cd NOME DA PASTA DESEJADA
    ```
3. Compile os arquivos .java de forma multipla:
    ```bash
    javac *.java
    ```
4. Execute o arquivo compilado:
    ```bash
    java NOME DO ARQUIVO
    ```

## 🎯 Objetivos

Este projeto foi desenvolvido para praticar:

- Estudar as estruturas de dados lineares;
- Revisitar conseitos de programação orientada a objetos;
- Aplicar algoritmos de inserção e remoção;
- Busca e percursos em estruturas;
- Trabalhar a complexidade de algoritmos;
- Boas práticas de desenvolvimento.
